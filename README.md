Project (Management System Project)

The project was designed as a simple management system that simulates and organizes a specific process within an organization. The main idea was to improve how data, tasks, or resources are managed in a structured and efficient way.

The system helps to:

Organize data in a structured manner
Improve decision-making processes
Reduce manual errors in management
Enhance efficiency in task distribution and workflow

The project also applies basic management concepts such as planning, organizing, and monitoring in a practical system.

📄 Paper (Research Report)

The paper is an academic report that explains the theoretical and managerial aspects of the project. It focuses on analyzing the problem and presenting a proposed solution from a management perspective.

It includes:

Introduction to the management problem
Importance of the proposed solution
Analysis of the current system
Description of the proposed system
Benefits of implementing the system
Conclusions and recommendations
🎯 Overall Objective

The main goal of both the project and the paper is to connect management concepts with practical implementation, showing how structured systems can improve organizational performance and efficiency.
